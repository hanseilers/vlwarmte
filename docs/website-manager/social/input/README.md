# Social Media Input

Zet hier de foto's en teksten voor de Social Media Agent.

## Hoe werkt het

1. Sleep foto's van projecten/werkzaamheden in deze map
2. Voeg optioneel een `briefing.txt` toe met context (project, locatie, bijzonderheden)
3. De Social Media Agent pakt deze automatisch op bij de volgende cyclus

## Naamgeving foto's

Gebruik beschrijvende namen:
- `project-groningen-woonhuis-2024.jpg`
- `installatie-schuimbeton-assen.jpg`
- `eindresultaat-leeuwarden.jpg`

## Briefing format (optioneel)

Maak een `briefing.txt` met:
```
Project: [naam/locatie]
Type: [woonhuis/appartement/utiliteit]
Diensten: [welke diensten zijn uitgevoerd]
Bijzonderheden: [wat is interessant aan dit project?]
Doelgroep post: [aannemers/particulieren/beide]
```
